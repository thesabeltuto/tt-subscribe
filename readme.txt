==== INSTALLATION ===

	1. Download this plugin, then upload, install and activate plugin to your wordpress site.
	2. Insert code to your desired theme page: 
		<?php if(function_exists('tt_subscribe')) { tt_subscribe(); } ?>
	3. Go to Dashboard -> Settings -> TT-Subscribe and change all settings according to your liking!
